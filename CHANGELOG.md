# Release Notes for Lens

## 1.0.0 - 2026-03-01

### Added
- Initial release of Lens — AI-powered image analysis, tagging, and search for Craft CMS assets.

#### Analysis & Tagging
- Alt text generation with confidence scoring.
- Suggested title generation for better asset organization.
- Long description generation for detailed image context.
- Semantic tag extraction with confidence scores.
- Dominant color palette extraction with percentages.
- OCR text extraction from images.
- Image quality scoring (sharpness, exposure, noise, overall).
- Focal point detection for smart cropping.
- EXIF metadata extraction (camera info, GPS coordinates, dates).
- Duplicate detection via perceptual hashing with similarity scores.

#### Content Detection
- Face and people detection (6 modes: none, people without faces, individual, duo, small group, large group).
- NSFW scoring with category breakdown.
- Watermark detection with type classification (text overlay, logo, stock provider).
- Brand/logo recognition for marketing assets.
- Stock photo provider identification.

#### AI Providers
- OpenAI GPT (GPT-5.2, GPT-5 Mini, GPT-5 Nano).
- Google Gemini (Gemini 2.5 Flash, Gemini 2.5 Flash Lite, Gemini 2.5 Pro).
- Anthropic Claude (Claude Sonnet 4.5, Claude Opus 4.5, Claude Haiku 4.5).
- Environment variable support for API key configuration.

#### Control Panel
- Dashboard with usage stats, cost projections, and quick actions.
- Asset Browser with 20+ filters, full-text search, and CSV export.
- Review Queue with three modes: Browse, Focus (keyboard shortcuts), and Bulk.
- Bulk Processing with real-time progress tracking and cost estimation.
- Analysis Panel in asset editor with inline editing for all fields.
- Inline editing with "Revert to AI" for any edited field.
- Configurable `requireReviewBeforeApply` mode for review-before-publish workflows.
- Keyboard shortcuts for rapid review (A=Approve, S=Skip, R=Reject).
- Settings page with provider selection, volume configuration, and workflow options.
- Logs page with filtering, retry capability, and detail inspection.
- Automatic processing on asset upload and file replace (configurable).
- Find Duplicates element action for asset indexes.
- Token usage tracking and cost estimation.

#### Asset Query Extensions
- `lensStatus()`, `lensContainsPeople()`, `lensConfidenceAbove()` / `lensConfidenceBelow()` — status, people, and confidence filters.
- `lensHasTags()` / `lensTag()`, `lensColor()` — tag and color filters.
- `lensNsfwFlagged()`, `lensHasWatermark()` / `lensWatermarkType()` — safety and watermark filters.
- `lensContainsBrandLogo()` / `lensDetectedBrand()`, `lensStockProvider()` — brand and stock filters.
- `lensHasDuplicates()`, `lensTextSearch()` — duplicate and full-text search.
- `lensQualityBelow()` / `lensSharpnessBelow()` / `lensExposureIssues()` — quality filters.
- `lensHasGpsCoordinates()` / `lensHasExifData()` — EXIF and GPS filters.
- 10 custom condition rules for element sources.

#### Console Commands
- `lens/process-all` — Process all unprocessed assets (with `--reprocess` option).
- `lens/process-volume` — Process a specific volume.
- `lens/scan-duplicates` — Scan for duplicate images.
- `lens/retry-failed` — Retry all failed analyses.
- `lens/validate` — Validate AI provider credentials.
- `lens/stats`, `lens/tag-stats`, `lens/color-stats` — View statistics.
- `lens/list-volumes` — List available volumes.

#### Twig Extensions
- `lens.getAnalysis()` — Access analysis data for any asset.
- `lens.getTagsForAnalysis()`, `lens.getColorsForAnalysis()` — Get tags and colors.
- `lens.getDuplicateCount()` — Check for duplicates.
- `lens.isAutoGeneratedTitle()` — Check if title was auto-generated by Craft.
